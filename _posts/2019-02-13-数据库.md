### docker安装启动
    - docker pull mysql
    - sudo docker run --name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql

### 误区
    - mysql -u root -p 命令修改默认密码
    - mysql 版本与connector版本要一致
